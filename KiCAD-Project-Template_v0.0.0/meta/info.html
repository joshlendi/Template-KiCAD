KiCAD Project Template

This project template is the basis for developing circuit boards in KiCAD 7.0. Various folders are created within the project to store documents, libraries, exports, etc.

Folder Structure

↳datasheet
↳export
↳images
↳libraries
↳symbol
↳footprint
↳meta
↳model
↳utilities

datasheet

Component and device datasheets

export

Schematics, gerber, other manufacturing files

libraries

Symbol and footprint libraries. Contains individual folders for each

model

3D model files for footprints, board, etc.

utilities

Various scripts and tools used in the project

Compatibility

This template is compatible with KiCAD 7.0
